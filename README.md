# termatrix

A terminal-based Matrix-like animation written in Go. It was just a bit of fun on an afternoon.

<p align="center">
  <img src="./media/matrix.gif" alt="Matrix" width="340" max-width="100%">
</p>

## Installation

```bash
go install github.com/jyecusch/termatrix@latest
```

## Usage

```bash
termatrix
```

Press `q` to quit.

## Development

```bash
make run

make build
```
